# Paper

`kv_hadamard.tex` — full technical report. Figures are pulled from `../figures/`
(via `\graphicspath`), generated by `../make_figures.py`.

## Compile

```bash
cd paper
pdflatex kv_hadamard.tex
pdflatex kv_hadamard.tex   # second pass resolves \ref / \cite
```

No external `.bib` — references are inline (`thebibliography`), so it compiles
with a stock TeX install (texlive-latex-base + texlive-latex-extra for
booktabs/hyperref). To regenerate figures first: `python ../make_figures.py`.

## Online (no local TeX)

Upload `kv_hadamard.tex` + the `figures/*.png` to [Overleaf](https://overleaf.com)
(New Project → Upload), keeping `figures/` one level up or adjust `\graphicspath`.
